LLM Agent for Concept Investing

Agent热度选股

这是一个把研报检索、LLM 文本理解、A 股候选池、组合构建和月度回测串起来的多节点选股框架。 当前主线是 Concept Agent：先读懂一个市场主题，再拆出产业链线索、找到相关公司、形成组合并留下可复盘证据。

代码入口：src/run.py 核心 Agent：src/agent/concept_agent.py 生成脚本：scripts/generate_project_poster.py 生成日期：2026-06-03
输入
主题 + 时间点单次模式输入一个主题和观察日期；月度模式逐月寻找当时最热的主题。
理解
先找材料，再让 LLM 概括每一步都只看 as_of 当日前可获得的信息，避免把未来新闻带进历史回测。
落地
从主题落到股票例如“AI算力链”会进一步落到服务器、芯片、光模块、PCB、液冷等公司线索。
复盘
结果不是一句话，而是一组文件每个节点都有 JSON 产物，方便回看当时为什么选、选了什么、哪里失败。

输入输出契约

框架的关键价值在于“把 LLM 放进可审计的契约里”：每个节点都有统一结构、引用来源、状态和错误字段。

运行输入

单次概念选股与月度回测共用同一套 Agent 基础设施，但调度层不同。

单次python -m src.run --concept "人形机器人" --as-of 2026-04-30
月度python -m src.run --monthly --start 2022-12-30 --end 2026-04-24
数据研报检索、公司资料、行情估值、Wind 指数和个股日线。

运行输出

产物按运行和月份落盘，既能复盘单个概念，也能重放历史净值。

Agent 在哪里发挥作用

下面尽量用“人话”描述每一环：它们共同完成从市场热点到股票组合的翻译工作。

1

找当月热门主题

先看近三个月研报和观点，找出市场正在反复讨论、且能落到股票上的主题。

例：2026-04 选出“AI算力链”和“锂电产业链”。
2

把大主题拆小

一个主题太大，Agent 会把它拆成几个更容易查证的方向。

例：“AI算力链”拆成服务器、光模块、AI芯片、PCB、液冷。
3

从方向找到公司

每个方向继续查材料，找到业务确实相关的 A 股公司，并记录为什么相关。

例：服务器方向可能出现中科曙光；光模块方向可能出现中际旭创。
4

合并成组合

把两个主题下的候选股票合并去重，按相关性排序，最多保留 30 只并做等权。

例：30 只股票时，大多数权重为 3% 或 4%，总权重保持 100%。
5

写复盘说明

最后把选择逻辑、主要风险和信心程度整理成报告，方便人工检查。

例：看好数据中心投资，但提示订单兑现、估值回落和竞争加剧风险。

代表性月份与原因推断

最佳超额2023-02-28：月收益 22.85%，超额 23.31%。主题为 AIGC与国产大模型、信创产业链。推断：ChatGPT 引发的 AI 主线在 2023 年初迅速扩散，软件、算力、信创相关股票弹性大；同期沪深300略跌，放大了超额。
强反弹2024-01-31：月收益 22.24%，超额 12.89%。主题为 AIGC应用落地、算力升级。推断：前期调整后的 TMT 与算力资产集中修复，AI 应用和国产算力叙事增强，组合相对宽基更偏高弹性成长。
最大拖累2023-06-30：月收益 -10.88%，超额 -15.36%。当月持仓 17 只。推断：AI/游戏/传媒链在前期上涨后回撤，风险偏好降温；而沪深300当期上涨 4.48%，主题组合和基准方向相反。
最新一期2026-04-24：选中 AI算力链、锂电产业链，30 只等权，月收益 -0.54%。推断：这是不完整持有期，回测逻辑近似同日买卖，收益主要体现换手成本；不宜和完整月份直接比较。
